# 商品上架 SEO 自動化_展示用 (MVP)

爬蟲抓回的陽春商品資料，自動**補圖、補敘述、生成 SEO/GEO 商品名與標籤**，透過**自建宜搭 (Yida) MCP**
寫回表單，再用 **n8n** 自動發社群貼文並回收成效。本頁為**靜態前端展示**，商品與數據皆為**虛構範例**。

> 📌 公開作品展示頁；不含任何金鑰、宜搭 token、表單 id 或真實商品/營運資料。

## 線上展示

直接以瀏覽器開啟 `index.html`；若資源載入異常：

```bash
python -m http.server 8000   # 開 http://localhost:8000
```

## 功能特色

- **流程動畫**：爬蟲商品 → 宜搭 J1 讀取 → AI 補圖/敘述/SEO → 寫回宜搭 → n8n 社群貼文 → 回收成效
- **原始 → 完成 前後對比**：選商品後一鍵 AI 處理，逐步補上圖片、SEO 名稱、列點＋GEO 敘述、標籤
- **社群自動化模擬**：發佈貼文並回收觸及/讚/分享/點擊數據
- 深色專業介面、響應式

## 在這個專案我實際做了什麼（高層說明）

- 🔌 **自建宜搭 MCP Server**：讓 AI 標準化讀取 J1 新品表單、回寫 J1 / J1A（insertOrUpdate / 最新版本）
- 🤖 **AI SEO / GEO 內容生成**：依爬蟲原始資料生成商品名、敘述（純文字＋HTML）、標籤，關鍵字前置
- 🖼️ **自動補圖 / 補敘述**：把缺圖缺敘述的陽春商品補成可上架內容
- ⚙️ **批次工具**：可配置、preflight 預檢、只更新差異、API 重試，落地中繼 JSON / 報表 / 統計
- 📣 **n8n 社群自動化**：上架完成自動發社群並回收觸及/互動/點擊成效寫回追蹤
- 📒 **品質管控**：不編造、不誇大、強制日誌、可追溯
