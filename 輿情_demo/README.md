# 台股輿情_MVP展示用

一個**互動式台股輿情分析儀表板**的輕量展示版（MVP）。
本頁為**靜態歷史快照**，純粹展示介面、互動與分析呈現，不做即時更新。

> 📌 這是公開的作品展示頁，僅保留前端介面與少量範例資料；不含任何系統設定、金鑰或營運細節。

## 線上展示

以瀏覽器直接開啟 `index.html` 即可；若分頁切換或字型載入異常，改用本機伺服器：

```bash
python -m http.server 8000
# 開 http://localhost:8000
```

## 功能特色

- **產業分析（族群燈號）**：全市場各產業情緒燈號與熱度排行
- **多分頁**：族群燈號、討論攀升、重點新聞、近期警示、熱議排行、個股精簡、個股詳情、成交量
- **個股搜尋**：輸入代號或名稱即時跳轉（`/` 鍵快速聚焦），支援鍵盤選取與 `#代號` 深連結
- **個股深度卡片**：多窗口（3／7／30／60 日）情緒分數、多空著色、相關新聞
- **AI 輔助情緒分數**：以分數與多空標示呈現市場情緒
- **產業群組展開／收合、響應式版面、深色主題**

## 展示資料範圍

| 項目 | 內容 |
|------|------|
| 主頁快照 | 單一交易日（2026-05-28）|
| 個股深度追蹤 | **24 檔**，橫跨 7 大產業：散熱、半導體大廠、IC設計、伺服器/代工、被動元件、記憶體、半導體材料 |
| 產業分析 | 全市場族群燈號（涵蓋所有產業情緒）|
| 更新方式 | 靜態快照，不即時更新 |

> 主頁的族群燈號／熱議榜等彙總區塊會列出較多個股與產業以呈現完整介面；
> 點選非上述 24 檔的個股時，深度跳轉會自然無作用，不影響展示。
