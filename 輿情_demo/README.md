# 台股輿情日報｜展示版（Beta）

一份**單日台股輿情＋籌碼快照報告**的公開展示版，以 [ECharts](https://echarts.apache.org/) 重建互動圖表，
呈現社群輿情、多因子選股、法人籌碼等面向的整合分析。
本頁為**靜態歷史快照**（2026-06-30），純粹展示介面、資料呈現與互動設計，不做即時更新。

> 這是公開的作品展示頁，僅保留前端介面與局部去識別化範例資料；不含任何系統設定、金鑰或營運細節。

## 線上展示

以瀏覽器直接開啟 `index.html` 即可；若圖表或字型載入異常，改用本機伺服器：

```bash
python -m http.server 8000
# 開 http://localhost:8000
```

## 功能特色

- **族群燈號 / 討論度攀升 / 近期警示**：全市場產業情緒燈號、近3天討論度變化、異常訊號提示
- **市場寬度、全市場熱議榜、國際輪動雷達（含美股連動）**：跨市場情緒與資金輪動視角
- **個股多因子表**：結合輿情分數與量化因子的多維度個股篩選表
- **法人籌碼**：外資／投信／自營商買賣超等籌碼面資料呈現
- **成交量排行、產業分析**：以 ECharts 繪製的排行與族群分析圖表
- **財經頻道每日精華、Threads 社群摘要**：多來源輿情彙整
- **英文財經術語子頁**（`english-stock-terms.html`）：CNBC 等真實英文財經報導選輯，搭配中文對照，做為英文財經閱讀輔助
- **深色主題、響應式版面**
